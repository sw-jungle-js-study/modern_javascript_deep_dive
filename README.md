## 👍 **진행 방식**
- 평일엔 1시간씩 책을 읽고, 토요일엔 읽은 내용을 바탕으로 설명 or 퀴즈를 푸는 시간을 갖습니다.
- 책을 읽다가 내용이 약간 부실하다 or 이해 안되는 것을 피드백과 함께 질문하는 시간을 갖습니다.
- 1회독 후, 타입스크립트 책으로 확장해보는 것 까지 생각하고 있습니다!
